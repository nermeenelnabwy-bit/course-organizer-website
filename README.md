# Course Organizer Website

## Overview
Course Organizer Website is a simple and responsive website designed to help students organize and browse their courses through a clean and user-friendly interface.

## Features
- Clean and modern user interface
- Responsive design
- Organized course sections
- Easy navigation between pages

## Technologies Used
- HTML5
- CSS

## Project Structure
```
Course Organizer Website/
│── index.html
│── contact.html
│── courses.html
|__ add.html
└── style.css
```

## Purpose
This project was developed as part of my university coursework to practice front-end web development using HTML and CSS.
